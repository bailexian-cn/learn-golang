# fmt.Sprintf

用于字符串拼接

```go
fmt.Sprintf("My name is %s", name)
```

## 字符串格式化

|      |                                                              |
| ---- | ------------------------------------------------------------ |
| %s   | string                                                       |
| %d   | 输出十进制                                                   |
| %b   | 输出二进制                                                   |
| %c   | 输出一个值的字符(char)                                       |
| %x   | 输出一个值的十六进制,每个字符串的字节用两个字符输出。打印的是一个值的16进制转型，例如int32(-1)输出的是-1，而不是0xffffffff |
| %f   | 输出浮点型数值                                               |
| %q   | 输出带双引号的字符串                                         |
| %p   | 输出一个指针的值                                             |
| %v   | 输出结构体的对象值                                           |
| %+v  | 输出结构体的成员名称和值                                     |
| %#v  | 输出一个值的Go语法表示方式                                   |
| %T   | 输出一个值的数据类型                                         |

## 对齐

|      |                                   |
| ---- | --------------------------------- |
| -5%s | 左对齐，长度小于5的字符串自动补齐 |
| 5%s  | 右对齐，长度小于5的字符串自动补齐 |
|      |                                   |
